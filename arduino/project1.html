Project #1: Blinking LED

Learning Goals: Arduino Basics and the IDE

In this project we'll get a light to blink on and off. You'll learn the basics of Arduino and how to make programs for it. We'll start by getting the light to blink and later modify the code to make it do different things using the same wiring.

Materials Required:

  • 2x Jumper Wires
  • 3x Connector Wires
  • 1x 220 Ohm Resistor
  • 1x LED
  • 1x Breadboard
  • 1x Arduino Uno

How to Build:

  1. Connect one end of the first jumper cable to the #9 socket on the Arduino and the other end to the (+) row on the breadboard.
  2. Link the (+) connection to a 220 Ohm resistor.
  3. Link the resistor to to longer positive end of an LED.
  4. Link the shorter negative end the LED to the (-) row.
  5. Connect the (-) row on the breadboard to the GND socket on the Arduino.

Code to Upload:

serial.begin(9600);
