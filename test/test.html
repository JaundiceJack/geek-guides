Getting to Know the Arduino IDE

Whether you want to make a drone army or make an led blink on and off, you'll need to program it. The makers of the Arduino foresaw your plight and made the Ardiuno IDE, which simplifies the process greatly. Head Here to pick it up.

Installing the IDE: Windows

Download setup.exe and run it. Follow the on-screen instructions.

Installing the IDE: Linux

Open your distro's package manager and search for Arduino. The Arduino IDE should be one of the first results. Note: When trying to upload code to the Arduino in Linux you may be presented with an error that looks like this:

To fix this run the following:

sudo usermod -a -G dialout yourusername
sudo chmod blah blah
